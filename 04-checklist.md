# 04 - Checklist da Fase 02 - Redes

Use este checklist para acompanhar o progresso nos estudos.

---

## Conceitos

- [ ] Sei explicar o que é uma rede
- [ ] Sei explicar o que é IP
- [ ] Sei explicar o que é porta
- [ ] Sei explicar a diferença entre TCP e UDP
- [ ] Sei explicar o que é DNS
- [ ] Sei explicar o que é HTTP
- [ ] Sei explicar o que é HTTPS
- [ ] Sei explicar o que é SSH
- [ ] Sei explicar o que é DHCP
- [ ] Sei explicar o que é NAT
- [ ] Sei explicar o que é sub-rede
- [ ] Sei explicar o que é gateway
- [ ] Sei explicar o que é firewall
- [ ] Sei explicar o que é `127.0.0.1`
- [ ] Sei explicar o que é `0.0.0.0`

---

## Comandos

- [ ] Executei `ip a`
- [ ] Executei `ip route`
- [ ] Executei `ping 8.8.8.8`
- [ ] Executei `ping google.com`
- [ ] Executei `traceroute google.com`
- [ ] Executei `dig google.com`
- [ ] Executei `nslookup google.com`
- [ ] Executei `curl -I https://google.com`
- [ ] Executei `ss -tulpn`
- [ ] Executei `nmap 127.0.0.1`
- [ ] Executei `nmap` no IP da minha VM
- [ ] Executei `tcpdump`

---

## Interpretação

- [ ] Identifiquei o IP da minha máquina
- [ ] Identifiquei minha rota padrão
- [ ] Identifiquei meu gateway
- [ ] Entendi se meu DNS está funcionando
- [ ] Identifiquei portas abertas
- [ ] Identifiquei serviços em LISTEN
- [ ] Diferenciei serviço em `127.0.0.1` e `0.0.0.0`
- [ ] Comparei resultado do `ss` com resultado do `nmap`
- [ ] Capturei tráfego básico com `tcpdump`
